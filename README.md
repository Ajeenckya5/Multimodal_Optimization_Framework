# Multimodal Optimization Framework for Logistics Planning

This project implements **OptiRoute**, a constrained multimodal routing framework that computes **cost–time optimal travel paths** across bus, train, and flight networks.

Routes are generated by solving an **exact Mixed-Integer Linear Program (MILP)** that enforces flow conservation, limits stopovers, and respects mode feasibility for geographically isolated destinations.

---

## Key Highlights
- Unified multimodal transportation network (bus / train / flight)
- Exact optimization (no heuristics)
- Cost–time tradeoff via a tunable composite objective
- Scalable per-destination formulation
- Interpretable, constraint-compliant itineraries

---

## Results
Optimized **2,040 routes** from a central hub across the U.S., achieving **negligible transportation cost relative to revenue** while maintaining operational feasibility.
